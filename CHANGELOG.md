# 更新日志

## v0.3.0 — 2026-06-01

完整 UI/功能重构，对标成熟 mihomo 客户端。

### 新增
- **侧栏重构**：应用头部、分组导航（概览/连接/策略/规则/配置/日志 · 引擎: 通用/网络/DNS/TUN/嗅探 · 实验室: 地图）、状态页脚、总开关。全局系统字体 (SF Pro + PingFang SC)。
- **概览仪表盘**：问候语、总下载/上传/连接数/访问目标、Metal 流量趋势、核心/应用内存、流量分布、策略组排行、流量时间轴、高频规则/热门域名/热门节点、客户端源 IP/热门进程/目标分类。
- **策略页**：类型图标、当前出口+延迟、点选切换、单组/全部测速、展开折叠。
- **规则页**：启用/禁用、编辑、上移/下移、删除、复制、添加（热重载）。
- **多配置管理**：本地配置卡片、导入远程订阅 / 添加本地、一键切换（持久化+热重载）、YAML 编辑。
- **网络页**：入站端口、IPv6/MPTCP/TCP 并发、访问控制（allow-lan/绑定/允许-拒绝 IP/认证/免认证）、系统代理开关。
- **通用页**：路由与连接、GEO 数据库、GEO 下载源、内核管理（正式版/Alpha 通道、检查更新、下载、重启）。
- **DNS/TUN/嗅探页**：可编辑配置表单 + DNS 解析测试 + Fake-IP 映射。
- **TUN / 系统代理总开关**：TUN 经 patch_config，系统代理经 networksetup（单次管理员授权）。
- **内核管理**：stable/alpha 通道、GitHub release 检查、darwin-arm64 内核下载。

### 引擎
- `patch_config` 配置深合并写入原语（校验 + 回滚）。
- 放宽 dev 覆盖，仅强制 TUN off，端口编辑可生效。
- 设置 mihomo home dir，providers/geodata/cache 正确解析。
